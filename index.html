컨설턴트 전용 · 로그인 후 서비스 선택 → 이력서 업로드 → 리포트 생성 → PDF 다운로드
※ 이 페이지는 정적 사이트용 “접근 게이트” 수준입니다. (완전 보안 로그인 아님)

HEDING Report Builder (Simple)

1) 서비스 선택 → 2) 이력서 업로드(PDF/DOCX) → 3) 리포트 자동 생성 → 4) 컨설턴트 코멘트 수정 → 5) PDF 다운로드

Step 1. 서비스 선택

※ HEDING 커리어 컨설팅 페이지 메뉴 기준

Step 2. 이력서 업로드

- PDF는 텍스트 레이어가 있어야 잘 추출됩니다(스캔본은 약함).
- DOCX는 문단 텍스트를 추출합니다.

AI 설정(컨설턴트용)

※ 키는 저장하지 않습니다(새로고침 시 유지 안 함).
※ 백엔드 없이 AI 자동 생성을 하려면 키가 필요합니다.

Step 4. 컨설턴트 코멘트

리포트 생성 후에도 언제든 수정 가능하며, PDF에 포함됩니다.
PDF는 화면의 리포트(자동 생성 + 코멘트 포함)를 그대로 저장합니다.

리포트 미리보기

서비스 선택 및 이력서 업로드 후 ‘리포트 자동 생성’을 누르세요.
PDF 1개 파일
아직 리포트가 생성되지 않았습니다.
Step 1~3를 진행하면 자동 생성된 결과가 여기에 표시됩니다.
